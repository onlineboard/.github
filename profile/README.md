We are **Onlineboard**, free multiplayer whiteboard with real-time collaboration support running in a web browser.

### Our fee plan includes:
- Real-time collaboration
- Dark mode
- Share by URL
- Export to PNG

We are happy to provide pro access to our tool to other OSS (open-source software) maintainers for free. We love open-source and want to support anyone who is developing open source project.
